# write-or-lose

A game called "Keep Writing" where your task is to write a short story but if you stop writing for 5 seconds all of your progress is deleted. Include a well designed little tutorial for new players. Include a roadmap for improvements and new features that can be viewed by the player.

## Tech stack

This project is built with React with shadcn-ui and Tailwind CSS.

- Vite
- React
- shadcn/ui

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/write-or-lose.git
cd write-or-lose
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
